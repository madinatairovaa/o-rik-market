# 🍑 O'rik Market

**O'rik Market** — bu Spring Boot texnologiyasi asosida yaratilayotgan zamonaviy backend loyihasi. Loyihaning asosiy maqsadi nafaqat dasturlashni o'rganish, balki jamoa bo'lib ishlash (teamwork) madaniyatini shakllantirishdir.

## 🚀 Loyiha Maqsadi
Ushbu loyiha kichik va o'rta hajmdagi do'konlar uchun mahsulotlar qoldig'ini boshqarish, savdo amallarini yuritish va hisobotlarni shakllantirishga xizmat qiladi.

## 🛠 Texnologiyalar
* **Java 17+**
* **Spring Boot 3.x**
* **Spring Data JPA** (Hibernate)
* **PostgreSQL** (Ma'lumotlar bazasi)
* **Gradle** (Loyiha menejeri)
* **Lombok** (Kodlarni qisqartirish uchun)
